#### ☆使い方☆  
- 「/wp-content/themes/<あなたのテーマのフォルダ>/」に「gbookapi.shortcode.php」をアップロードしてください。  
- 使用中のテーマの「functions.php」に一行書き加えてください。  
  
```PHP
<?php
  // Some code...
  include("gbookapi.shortcode.php");

```
